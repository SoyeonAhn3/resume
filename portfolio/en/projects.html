AI Stock Analyzer

Full-Stack GitHub

A multi-agent stock analysis system where 5 AI agents analyze news, financials, and macro indicators in parallel, cross-validate each other, and deliver investment recommendations.

Motivation

When analyzing stocks, having to check prices, financials, news, and macro data across multiple platforms felt inefficient. Rather than querying a single AI, I wanted to design and implement a multi-agent architecture where 5 specialized agents analyze from different perspectives and cross-validate each other's findings.

Key Features

Quick Look

Instantly retrieve real-time prices, candle charts, financial metrics, and technical indicators (MACD, Bollinger Bands) within 1 second.

5-Agent Deep Analysis

News, Data, Macro, Cross-Validator, and Analyst agents run in parallel, cross-validate findings, and deliver BUY/HOLD/SELL recommendations.

7 APIs + Fallback Chain

Integrates yfinance, Finnhub, Twelve Data, Finviz, FMP, FRED, and Claude API. Automatically tries alternative sources when the primary API fails.

Anti-Hallucination Design

All numbers come from APIs only; AI handles interpretation only. Role separation ensures reliability in the financial domain.

Problem Solving

Problem

Frequent 403 errors from external data APIs' free plan limits causing analysis to break

Solution

Built a multi-fallback chain (yfinance → Finviz → FMP) and applied Graceful Degradation per agent, so the remaining analysis continues even when one source fails

React TypeScript FastAPI Claude API SQLite yfinance Lightweight Charts asyncio

Workflow Generator

Live Demo GitHub

A web app that automatically generates diagrams and Word manuals from business process descriptions.

Motivation

Visualizing business processes and managing them as documents was time-consuming and tedious. I thought if there was a tool that could automatically create diagrams and manuals from text input, it would significantly reduce the documentation burden.

Key Features

AI Auto-Structuring

Claude API analyzes text descriptions and automatically converts them into department, group, process, and step structures.

SwimLane + LinearFlow

Renders SwimLane diagrams showing department roles at a glance and LinearFlow diagrams showing step-by-step flows in SVG.

Auto Word Manual

Generates .docx documents with cover page, summary table, step details, and images directly in the browser.

Drag & Drop Editing

Step reordering with @dnd-kit, parallel branching (colIndex) support for intuitive editing of complex processes.

Architecture

User Input
Text / Direct Input
Netlify Functions
Claude API Proxy
React SPA
SVG Diagrams + Editor
Output
Diagram + Word .docx
LocalStorage (text) IndexedDB (images)

Problem Solving

Problem

AI structures text into sequential steps, but real workflows often have parallel flows like "A and B run simultaneously" that can't be represented as simple lists

Solution

Assigned colIndex (column number) to each step to auto-group same-column steps as parallel branches, with @dnd-kit drag-and-drop for intuitive order and branch editing

React 18 Vite SVG Claude API Netlify Functions docx @dnd-kit IndexedDB

AI Production Monitor

🚧 In Progress GitHub

A fully automated system that monitors production data hourly, detects anomalies, and uses AI to analyze root causes and send alerts.

Motivation

Working in manufacturing IT operations, I realized the need for a system that quickly detects production anomalies and enables immediate response. I designed an End-to-End automation system where AI interprets the root cause of anomalies and detailed analysis can be reviewed through Power BI dashboards.

Key Features

Config-driven Detection

Detection rules defined in rules.json, executed by engine.js. 8 rules x 3 severity levels. Only modify rules to apply to new data.

AI Agent Daily Report (Tool Use)

AI autonomously queries and analyzes data with 4 tools. Auto-adjusts report depth: brief summary on quiet days, detailed analysis + weekly comparison on critical days.

4-Level Security Strategy

Provides basic/masking/template/local LLM options so organizations can adjust external data transmission levels to match their security policies.

Power BI Dashboard

3-page layout: real-time status, AI insights, long-term trends. Utilizes Key Influencers, Smart Narrative, and other AI visuals.
