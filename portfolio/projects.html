AI Stock Analyzer

Full-Stack GitHub

5개 AI 에이전트가 뉴스·재무·거시경제를 병렬 분석하고 교차 검증하여 투자 의견을 제시하는 멀티에이전트 주식 분석 시스템

개발 동기

주식 투자 시 여러 플랫폼에서 시세, 재무, 뉴스, 매크로 데이터를 개별 확인해야 하는 번거로움이 있었습니다. 단일 AI에 질문하는 방식이 아닌, 역할이 다른 5개 에이전트가 각자의 관점에서 병렬 분석하고 서로를 교차 검증하는 멀티에이전트 아키텍처를 직접 설계하고 구현해보기 위해 개발을 시작했습니다.

핵심 기능

Quick Look

티커 입력 즉시 실시간 시세, 캔들 차트, 재무 지표, 기술 지표(MACD·볼린저밴드)를 1초 내 조회합니다.

5-Agent Deep Analysis

News, Data, Macro, Cross-Validator, Analyst 5개 에이전트가 병렬 분석 후 교차 검증하여 BUY/HOLD/SELL 의견을 제시합니다.

7개 API 통합 + 폴백 체인

yfinance, Finnhub, Twelve Data, Finviz, FMP, FRED, Claude API를 통합. 1순위 API 실패 시 자동으로 대체 소스를 시도합니다.

AI 환각 방지 설계

숫자는 API에서만 가져오고, AI는 해석만 담당. 금융 도메인의 신뢰성 확보를 위한 역할 분리 설계.

Problem Solving

문제

외부 데이터 API의 무료 플랜 제한으로 403 에러가 빈번히 발생하여 분석이 중단되는 문제

해결

Finviz 래퍼를 추가하여 다중 폴백 체인(yfinance → Finviz → FMP)을 구성하고, 에이전트별 부분 실패 허용(Graceful Degradation) 구조를 적용하여 하나의 소스가 실패해도 나머지 분석이 계속 진행되도록 설계

React TypeScript FastAPI Claude API SQLite yfinance Lightweight Charts asyncio

Workflow Generator

Live Demo GitHub

업무 프로세스를 입력하면 다이어그램과 Word 매뉴얼을 자동으로 생성하는 웹 앱

개발 동기

회사에서 업무 프로세스를 시각화하고 문서로 관리하는 과정이 번거롭고 시간이 많이 소요되는 것을 보며, 이를 효율적으로 개선할 방법이 없을까 고민하게 되었습니다.
텍스트로 프로세스를 입력하면 자동으로 다이어그램과 매뉴얼이 만들어지는 도구가 있다면 문서화 부담을 크게 줄일 수 있겠다는 생각에서 개발을 시작했습니다.

핵심 기능

AI 자동 구조화

텍스트로 작성한 업무 설명을 Claude API가 분석하여 부서, 그룹, 프로세스, 단계 구조로 자동 변환합니다.

SwimLane + LinearFlow

부서별 역할을 한눈에 보여주는 SwimLane 다이어그램과 단계별 흐름을 보여주는 LinearFlow를 SVG로 렌더링합니다.

Word 매뉴얼 자동 생성

표지, 요약 테이블, 단계별 상세 설명, 이미지가 포함된 .docx 문서를 브라우저에서 바로 생성합니다.

드래그앤드롭 편집

@dnd-kit으로 구현된 단계 순서 변경, 병렬 분기(colIndex) 지원으로 복잡한 프로세스도 직관적으로 편집합니다.

Problem Solving

문제

사용자가 프로세스를 텍스트로 입력하면 AI가 구조화해주지만, 실제 업무는 "A와 B가 동시에 진행" 같은 병렬 흐름이 많아 단순 순차 목록으로는 현실을 표현할 수 없었음

해결

각 단계에 colIndex(열 번호)를 부여하여 같은 열의 스텝을 자동 병렬 그룹으로 처리하고, @dnd-kit 드래그앤드롭으로 순서·분기를 직관적으로 편집할 수 있도록 설계

React 18 Vite SVG Claude API Netlify Functions docx @dnd-kit IndexedDB

AI Production Monitor

🚧 개발 중 GitHub

생산 실적 데이터를 1시간 간격으로 자동 모니터링하고, 이상 감지 시 AI가 원인을 분석하여 알림을 보내는 완전 자동화 시스템

개발 동기

제조업 IT 운영 업무를 하며, 생산 현장의 이상을 빠르게 감지하고 담당자가 즉시 대응할 수 있는 시스템이 필요하다는 것을 체감했습니다. 단순 알림이 아니라 AI가 이상의 원인을 해석해주고, Power BI 대시보드에서 상세 분석까지 확인할 수 있는 End-to-End 자동화 시스템을 설계하게 되었습니다.

핵심 기능

Config-driven 이상 탐지

rules.json에 탐지 룰을 정의하고 engine.js가 범용으로 실행합니다. 8가지 탐지 × 3단계 심각도 분류. 새 데이터 적용 시 룰만 수정하면 됩니다.

AI Agent 일일 리포트 (Tool Use)

AI에게 4개 도구를 제공하여 상황에 맞게 자율적으로 데이터를 조회·분석합니다. 조용한 날은 간단 요약, 심각 건이 있는 날은 상세 분석 + 지난주 비교까지 리포트 깊이를 자동 조절합니다.

4단계 보안 전략

기본/마스킹/템플릿/로컬 LLM 단계별 선택지를 제공하여 조직의 보안 정책에 맞게 외부 데이터 전송 수준을 조정 가능합니다.

Power BI 대시보드

실시간 현황, AI 인사이트, 장기 트렌드 3페이지 구성. Key Influencers, Smart Narrative 등 AI 시각화 활용.
