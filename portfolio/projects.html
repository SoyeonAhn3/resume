Workflow Generator

Live Demo GitHub

업무 프로세스를 입력하면 다이어그램과 Word 매뉴얼을 자동으로 생성하는 웹 앱

개발 동기

회사에서 업무 프로세스를 시각화하고 문서로 관리하는 과정이 번거롭고 시간이 많이 소요되는 것을 보며, 이를 효율적으로 개선할 방법이 없을까 고민하게 되었습니다.
텍스트로 프로세스를 입력하면 자동으로 다이어그램과 매뉴얼이 만들어지는 도구가 있다면 문서화 부담을 크게 줄일 수 있겠다는 생각에서 개발을 시작했습니다.

핵심 기능

AI 자동 구조화

텍스트로 작성한 업무 설명을 Claude API가 분석하여 부서, 그룹, 프로세스, 단계 구조로 자동 변환합니다.

SwimLane + LinearFlow

부서별 역할을 한눈에 보여주는 SwimLane 다이어그램과 단계별 흐름을 보여주는 LinearFlow를 SVG로 렌더링합니다.

Word 매뉴얼 자동 생성

표지, 요약 테이블, 단계별 상세 설명, 이미지가 포함된 .docx 문서를 브라우저에서 바로 생성합니다.

드래그앤드롭 편집

@dnd-kit으로 구현된 단계 순서 변경, 병렬 분기(colIndex) 지원으로 복잡한 프로세스도 직관적으로 편집합니다.

Code Review Pipeline

GitHub

4개 AI 에이전트가 순차적으로 코드를 리뷰하고 점수를 매기는 멀티 에이전트 시스템

개발 동기

AI 자동화 프로젝트를 빠르게 반복하면서 보안·성능 측면의 체크포인트를 체계적으로 관리할 필요를 느꼈습니다.
보안, 성능, 코드 품질을 각각 전문적으로 검토해주는 시스템이 있다면 이런 문제를 사전에 방지할 수 있겠다는 생각에서 개발하게 되었습니다.

핵심 기능

4-Agent 순차 분석

Security, Performance, Code Quality, Summary 에이전트가 순서대로 실행되며, 이전 에이전트의 분석 결과를 컨텍스트로 전달받습니다.

교차 반론 (Cross-Review)

후속 에이전트가 이전 에이전트의 제안에 대해 agree/caution/disagree 의견을 제시하여 다각적 관점의 리뷰를 제공합니다.

100점 스코어링

Critical(-15), Warning(-5), Info(-1) 기준으로 점수를 산출하고, Top 3 수정 우선순위를 자동으로 제안합니다.

GitHub PR 연동

PR URL을 입력하면 변경된 파일의 diff를 자동으로 추출하여 리뷰합니다. 파일 직접 업로드도 지원합니다.

아키텍처

Input Code / PR URL / File
Agent 1 Security
Agent 2 Performance
Agent 3 Quality
Agent 4 Summary + Score
Output Report + Markdown

각 에이전트는 이전 에이전트의 결과를 누적 전달받아 교차 검증합니다. Tool Use로 JSON 구조를 강제하여 파싱 실패율을 최소화하는 것을 목표로 설계했습니다.

AI Consulting Agent

GitHub

MS 업무자동화 방안을 3개 AI가 동시에 분석하고 비교 검증하는 CLI 컨설팅 도구

개발 동기

Microsoft 생태계를 활용한 업무 자동화 요청이 많았는데, Power Automate, Office Script, Power Query 등 선택지가 다양하다 보니 어떤 방법이 가장 적합한지 판단하기가 쉽지 않았습니다. 또한 사용자 요청과 해결 방안을 체계적으로 문서화할 필요성을 느끼면서, 요구사항 분석부터 솔루션 비교, 산출물 생성까지 한 번에 처리할 수 있는 도구를 만들게 되었습니다.

핵심 기능

Quick / Deep 모드

Quick 모드는 Claude 단독 분석으로 빠른 결과를, Deep 모드는 Claude + Codex + Gemini 3개 AI 병렬 분석으로 교차 검증된 결과를 제공합니다.

재컨설팅 시스템

조건 변경(A), 솔루션 변경(B), 설명 요청(C) 3가지 유형으로 재컨설팅을 지원하며, 세션 상태를 파일로 관리하여 이력이 보존됩니다.

ROI 예측

자동화 전후 소요 시간을 기반으로 보수적/중립/적극적 3가지 시나리오의 ROI를 산출합니다.

다중 산출물

컨설팅 보고서(.txt), Excel 리포트(.xlsx), Power Automate Flow 설계서까지 자동으로 생성합니다.

아키텍처

Step 1 요구사항 파싱
Step 2 Scope Gate
Step 3 AI 분석 (3-AI)
Step 4 리스크 2-Pass
Step 5 산출물 생성

MS Learn 자동 검색으로 솔루션 지원 여부를 검증하고, 더 이상 지원되지 않는 제품은 Blocklist를 통해 자동으로 제외됩니다.

Notion Notes

GitHub

학습 자료를 AI로 분석하고, 유용한 답변만 골라 Notion에 자동 저장하는 웹 앱

개발 동기

학습한 내용을 AI와의 대화를 통해 정리하고, 유용한 답변만 골라 Notion에 바로 저장하는 흐름을 만들고 싶었습니다. 매번 AI 응답을 수동으로 복사하여 Notion에 붙여넣는 과정이 비효율적이라 느꼈고, 파일 업로드부터 분석, 저장까지 한 곳에서 처리할 수 있는 도구를 개발하게 되었습니다.

Demo screenshot coming soon

핵심 기능

멀티 파일 업로드

PDF, 이미지, Word, 텍스트 파일을 드래그앤드롭으로 업로드하고, Gemini File API로 토큰을 절약하며 처리합니다.

멀티 AI 대화

Gemini와 OpenAI 모델을 자유롭게 전환하며 멀티턴 대화를 진행합니다. API 키 유무에 따라 UI가 자동으로 대응합니다.

선택적 저장

대화 중 유용한 AI 응답만 체크박스로 선택하여 Notion에 저장합니다. 전체가 아닌 필요한 부분만 골라 저장할 수 있습니다.

Notion 계층 연동

Page → Database → Item 계층 구조를 지원하며, 새 항목 생성과 기존 항목에 내용 추가(append)가 가능합니다.
